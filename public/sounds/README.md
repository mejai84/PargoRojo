# Archivos de Sonido para Notificaciones

## Archivos Necesarios

### 1. notification.mp3
- **Uso**: Notificaciones generales (clientes, alertas suaves)
- **Características**: Sonido suave y agradable
- **Volumen**: 50%
- **Duración recomendada**: 1-2 segundos

### 2. new-order.mp3
- **Uso**: Nuevos pedidos en cocina (admin)
- **Características**: Sonido más fuerte y llamativo
- **Volumen**: 70%
- **Duración recomendada**: 2-3 segundos

## Instalación

1. Descarga los archivos MP3
2. Renómbralos a:
   - `notification.mp3`
   - `new-order.mp3`
3. Colócalos en esta carpeta: `public/sounds/`
