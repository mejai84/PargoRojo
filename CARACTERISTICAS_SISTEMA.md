# Documentación de Módulos Administrativos - Pargo Rojo

Este documento detalla las características principales del sistema de administración desarrollado para el restaurante **Pargo Rojo**, diseñado para optimizar la operación y mejorar la experiencia del cliente.

---

## 1. Página Principal (Landing Page)
La cara pública del restaurante, optimizada para atraer clientes y facilitar la navegación.
- **Diseño Premium**: Interfaz moderna y responsive adaptada a cualquier dispositivo.
- **Menú Digital**: Visualización interactiva de categorías y platos con precios actualizados.
- **Acceso Rápido**: Botones directos para realizar pedidos y reservas.
- **Historia y Ubicación**: Sección dedicada a la identidad del restaurante y su ubicación.

## 2. Módulo de Cocina (KDS - Kitchen Display System)
Centraliza la preparación de pedidos para una comunicación eficiente entre sala y cocina.
- **Gestión en Tiempo Real**: Recepción inmediata de pedidos realizados desde la web.
- **Estados de Pedido**: Control visual de platos (Pendiente, En Preparación, Listo).
- **Priorización**: Organización automática por orden de llegada.
- **Optimización de Tiempos**: Reducción de errores en la comunicación manual.

## 3. Gestión de Pedidos y Caja
Control total sobre las transacciones y el historial de ventas.
- **Listado Detallado**: Registro de todos los pedidos con su respectivo detalle de productos.
- **Seguimiento de Pagos**: Identificación de pedidos pagados y pendientes.
- **Filtros Avanzados**: Búsqueda por fecha, estado o cliente.
- **Reportes de Venta**: Visualización de ingresos y rendimiento del restaurante.

## 4. Sistema de Reservas
Módulo diseñado para gestionar la afluencia de clientes de manera organizada.
- **Calendario de Reservas**: Visualización clara de fechas y horarios ocupados.
- **Información del Cliente**: Registro de datos de contacto y preferencias.
- **Confirmación Automática**: Sistema para validar o rechazar reservas con notificaciones.
- **Optimización de Espacio**: Mejor planificación del aforo del restaurante.

## 5. Gestión de Inventario (Extra)
Control preciso de los insumos y productos disponibles.
- **Control de Stock**: Actualización automática de existencias tras cada venta.
- **Alertas de Bajo Stock**: Notificaciones para reposición de insumos críticos.
- **Gestión de Proveedores**: Directorio de suministros vinculados a productos.
- **Costeo de Recetas**: Análisis de rentabilidad por cada plato del menú.

---

Este sistema integral permite a **Pargo Rojo** operar con estándares de alta eficiencia, garantizando que cada pedido y reserva se maneje con la máxima precisión profesional.
